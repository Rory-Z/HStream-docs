# 安装

HStreamDB Java SDK 已经在 Maven 发布，查看 hstreamdb-java (opens new window).

# Maven

Maven 用户可以这样使用 HStreamDB 客户端:


<dependencies>
  <dependency>
    <groupId>io.hstream</groupId>
    <artifactId>hstreamdb-java</artifactId>
    <version>0.7.0</version>
  </dependency>
</dependencies>
